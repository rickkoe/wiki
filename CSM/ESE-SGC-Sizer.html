Calculating CSM Memory Requirements (ESESizer)

  • Sizing is not required for SGCsizer
  • Default heap size: 1280MB
  • java heap location(Xmxm parm in jvm.options file)   
    • window- install dir/liberty/wlp/usr/servers/csmServer/jvm.options
    • zOS - path_prefix/opt/IBM/CSM/wlp/usr/server/csmServer/jvm.options
  • Java heap size DS8000 capacity
    • 4 GB ≤ 20 TiB
    • 8 GB ≤ 60 TiB
    • 16 GB ≤ 60 TiB
  • Calculation for estimation of Optimal heap size - device memory usage (Mb) = Number of devices * 66
    • Nonsession processing = 512 MB (required) ------------ > 512 MB 
    • Non-safeguarded Copy sessions = 256 MB / 25,000 pairs (# of pairs/copy sets times the # of role pairs in the session)  ------ >  15000 vols x 3 roles = 45000 
      • need 512 MB to cover non-sgc sessions  
    • Safeguarded Copy sessions = (# of volumes)  x  (# of backups “ie - retention”)  x .8 KB   ---- > 2000vols & 250 bckups -  2000 x 250 x .8 = 400 MB
  • Total =   Nonsession processing + Non-safeguarded Copy sessions + Device Memory Usage
    • 512 MB + 512 MB + 400 MB = 1,424 MB
