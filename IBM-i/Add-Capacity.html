1. Determine how much capacity to add.  What is the current volume size for the LPAR?  How many volumes need to be added to satisfy the capacity needs.
  2. Map volumes to host(s).  Ensure you are not mapping more than 127 volumes per host on the FlashSystem/DS8000.
  3. Initialize and format the disk units  From the Use Dedicated Service Tools (DST) display, select Work with disk units > Work with disk unit recovery > Disk unit problem recovery procedures > Initialize and format disk unit. Select the disk unit you want to initialize and format. Press F10 on the confirmation display.
  4. Replicate the volumes
  5. Add volumes to the ASP
  6. Start balance
