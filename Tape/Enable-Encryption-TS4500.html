Setting up Key servers on TS4500 using TLS1.2 isn't very well documented.  Here's how we got it to work at Prime:

  • Go to Settings → Security → Secure Communications and set to On (you will lose browser access for about 5 minutes.  It gives a timer, but takes way longer to come back than that).
  • Go to the GKLM server, Advanced Configuration → System Certificates and download the key serving cert in base 64.
  • Open the cert in notepad ++, ctrl-H, regular expression button, find \r and replace with nothing.  Save as a .pem file
  • Go to Settings → Security → Encryption Key Servers
    • Click on one of the key servers, Actions → Modify
    • Change security to TLS 1.2 and set the port to 1441
    • Import the .pem file that was exported from GKLM and modified in the earlier step.
  • Run Diagnostics on the key server to ensure full key path is working.
