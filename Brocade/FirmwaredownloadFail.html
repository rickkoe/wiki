FOS version known issue with FOS v9.1.1- Firmwaredownload Fail

Ticket will need to be opened with support to resolve as service access is needed. 

******Steps will need to be run on both CPs- start with CP0)****

1.) Get response token

    a. Logon on as maintenance id (def password)

    b. Change password 'userconfig --change maintenance -e yes'

    c. Generate token 'serviceshell --generate'

    d. Show license 'license --show lid'

The above will be needed for support to generate response token.  

 

2.) Cleanup and reinit package database  

    a. Logon on as maintenance id

    b. Enable serviceshell 'serviceshell --authenticate "response token"'

    c. Activate serviceshell 'serviceshell --activate'  

    d. List files in image directory 'ls /mnt/var/images/'

    e. Remove images from directory 'rm -rf /mnt/var/images/*'

 

3.) Repeat step 1 & 2 for standby CP.  

4.) Reattempt firmwaredownload.
