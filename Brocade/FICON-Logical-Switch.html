Creating a FICON Logical Switch

1. Determine new Fabric ID

2. List config 
lscfg --show

4. Create Fabric -

lscfg –create 1 -ficon

5. Go to newly created fabric 

setcontext xx'

6. Configure domain???

7. Enable switch

switchenable

8. Add ports to fabric

lscfg --config 1 -port 0

lscfg --configure 1 -port 44

9. dlsshow??

10. Verify/Enable iodset

iodshow 

iodset    enables 

11. Setup secpolicy

12. Bound ports - bind is mostly for Ficon

portaddress --bind 0 0x0000

portaddress --bind 0 0xc000

13. Enable ports

14. Apply secpolicy 

secpolicyshow - show 

secpolicyadd sec_policy, “WWNN” 

secpolicyactivate

 15. Configure zonning

zoneadd <zonename>, "23,0"

16. Enable cfg

cfgenable <cfg name>
