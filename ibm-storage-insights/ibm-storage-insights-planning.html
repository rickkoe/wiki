If you’re an IBM Spectrum Control customer you’re eligible for IBM Storage Insights for IBM Spectrum Control—monitoring that delivers even more information about your IBM storage performance, capacity, and configuration metadata and includes alerts for storage systems requiring attention.

Getting Started

Full instructions can be found on IBM Knowledge Center

  1. Deploy one or more metadata collectors (for redundancy or to optimize collection from data centers in different locations).
    • Provide at least 1 GB of RAM and 3 GB of disk space
    • Operating System Options:
      • Windows Server 2012 and later
      • AIX versions 7.x and later on POWER6 or later systems
      • x86-64 RHEL6 and later
      • CentOS 6 and later
      • PPC64LE: POWER8 with RHEL7
  2. Allow outbound communication for metadata collector VM on port 443 using TCP
  3. If not already done, Register for IBM Storage Insights
  4. Wait up to 24 hours for your instance to be created
  5. Log in to your instance and download/deploy metadata collectors
  6. Add your storage to IBM Storage Insights

Security Considerations

IBM Storage Insights Security Guide

  • Firewall Port 443 (outbound) communication required for metadata collector
  • #IBMCloud service
  • ISO/IEC 27001/27017/27018 ISM certified
  • Communication is one way, encrypted and compressed
  • Data at rest is AES 256-bit encrypted
  • Only metadata about your storage is collected
  • Personal, identity, and application data are never accessed
  • HIPAA / Blue Diamond ready
  • Meets the requirements of GDPR
